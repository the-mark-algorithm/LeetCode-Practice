# LeetCode Practice

This repository is a personal Java workspace for practicing LeetCode problems and exploring multiple solution strategies.

## Purpose

The goal of this project is to:
- solve problems from different difficulty levels
- keep each problem in its own class
- compare multiple implementations for the same problem
- practice clean, readable Java code

## Project Structure

- src/easy/ - solutions for easy problems
- src/medium/ - solutions for medium problems
- src/hard/ - reserved for harder problems in the future
- src/common/ - shared data structures and reusable code
- src/utils/ - small helper utilities and supporting code
- bin/ - compiled Java output (generated locally)

## Style

Each problem file generally follows the same pattern:
- a `Solution` interface
- one or more `SolutionN` classes
- a `main` method for simple local input/output

The main focus is on learning and experimentation, so the structure is intentionally simple and easy to extend.
